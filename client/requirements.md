## Packages
framer-motion | Essential for complex scroll animations and interactive UI elements
lucide-react | Standardized iconography across the application
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge tailwind classes without style conflicts

## Notes
- Single page scroll design for a seamless flow, all sections are integrated into Home.
- Color theme is strictly Light Mode (White & Dark Blue / Slate-900). Dark mode is disabled.
- Images from attachments are used as visual references; fallback icons/UI elements are used to ensure the app works beautifully regardless of static asset availability.
- All endpoints rely on the shared/routes API contracts. Hooks are fully typed.
